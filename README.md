# Predictive-Control-System-Greece4.0
This repository documents the code to generate RBFNNs as predictive models in a control framework, developed for project "Greece4.0: Network of Excellence for the Development, Dissemination and Application of Digital Transformation Technologies in the Greek Manufacturing Industry".
